# SCANNING 3D | Premium Corporate Website

## 📌 About the Project
SCANNING 3D is a premium multilingual landing page for a company that provides high-precision 3D scanning and space measurement services using Leica equipment.

The site is designed with an emphasis on strict, minimalist design (Apple-style), typography and smooth animations. The main task of the project is to convert visitors (architects, interior designers and furniture manufacturers) into leads by demonstrating expertise and accuracy.

## 🚀 Key features
- Multilingualism: Full support for 3 languages ​​(Ukrainian, Russian, English) with seamless switching.
- Responsive design: Pixel-perfect layout for desktops, tablets and mobile devices (Mobile-first approach + use of clamp() for adaptive typography).
- Advanced CSS Animations: Smooth scrolling (reveal animations), interactive hover effects for project cards.
- Video Integration: Use of auto-playing, optimized mp4 videos in project cards to visually demonstrate 3D models.
- Interactive Slider: Custom slider for the section with technical specifications of the equipment.
- Lead Generation: Integrated contact capture form.

## 🛠 Tech Stack
- HTML5: Semantic markup, SVG icon integration.
- CSS3: Flexbox, CSS Grid, CSS Animations, Media Queries, CSS Variables for theme control.
- JavaScript (Vanilla JS): Slider logic, burger menu, scrolling (Intersection Observer), form processing.
- Fonts and Icons: Google Fonts (Inter), FontAwesome.
